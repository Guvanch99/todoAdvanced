
# Todo app

This project was generated via vite.



## Run Locally

Clone the project

```bash
  git clone https://github.com/Guvanch99/todoAdvanced
```

Go to the project directory

```bash
  cd todoAdvanced
```

Install dependencies

```bash
  yarn install
```

Start the server

```bash
  yarn run dev
```

## Deployment

To deploy this project run

```bash
  yarn run build
```
